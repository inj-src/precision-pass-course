The Server Shape

The Python backend is a small FastAPI app that owns data, recognition, attendance, leave, and payroll.

Think of the server as the front desk ledger

The server receives requests from the browser, validates them, calls helper modules, and returns structured JSON.

It starts with python3 -m uvicorn CV.server:app --reload from package.json:7 or CV/README.md:8-10.

Server actors talk like this

API
FastAPI

I receive the URL, method, body, query params, and uploaded files.

P
Pydantic

I check that incoming data matches the expected shape before route logic trusts it.

D
Data Layer

I read and write employees, attendance, and leaves from JSON files.

CV
Vision Layer

I manage detector state, training samples, recognition events, and model thresholds.

UI
Frontend

I consume JSON responses and update screens.
